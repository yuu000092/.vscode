# 要件定義

### 概要
テーマ「Todoリスト」
自己タスクの追加・編集・削除・一覧表示を可能とするプログラムを作成

### 機能要件
- タスクの追加・編集・削除・一覧表示
- タスク追加画面で「終了」と入力すると保存されてアプリが閉じる(終了というタスクは入力不可)
- タスクの上限は50件まで

### 非機能要件
- 閲覧画面は表形式の表示

### システム構成・外部連携
#### システム構成
##### アプリケーション
- メインメニュー
- タスクの登録
- タスク一覧
- タスク編集
- タスク削除

テキストファイル(.TXTファイル)
- 入力タスクの書き込み
- 一覧を選択した際に.TXTファイルの読み込み

```mermaid
flowchart LR
    アプリケーション <--> テキストファイル
```
### 外部連携
なし
