## 勤怠管理システムの要件定義

- 起動後、勤怠管理システムと現在時刻表示する
- 半角数字1～5に、上から出勤・退勤・編集・一覧・終了と表示
- ケースは下記に記述
 [1.出勤]
 - 前提条件：ステータスが未出勤
 - 例外条件：ステータスが出勤・退勤状態
 - エラーメッセージ：「本日出勤済みです」
 - 処理：ステータスを「出勤」に変更し、現在時刻をCSVファイルに保存
 - 状態遷移：未出勤→出勤
 - 保存形式：保存先ファイル(ID.csv)に追記(ID , 出勤時間)
  [2.退勤]
 - 前提条件：ステータスが出勤
 - 例外条件：ステータスが未出勤・退勤状態
 - エラーメッセージ：
    - 未出勤：「出勤記録がありません」
    - 退勤：「本日退勤済みです」
 - 処理：ステータスを「退勤」に変更し、現在時刻、休憩時間をCSVファイルに保存
 - 状態遷移：出勤→退勤
 - 保存形式：保存先ファイル(ID.csv)に追記(退勤時間 , 休憩時間)　 
  [3.編集]
 - 前提条件：CSVファイルに出勤時間、退勤時間が記録されている
 - 例外条件：
    - CSVに出勤時間、退勤時間が記録されていない
    - 半角数字以外の入力、および00:00～23:59以外の入力
 - エラーメッセージ：
    - 記録無し：「勤怠記録がありません」
    - 不正入力：「正しく入力して下さい」
 - 処理：現在、CSVファイルに書き込まれている内容を新たな時刻に書き換える
 - 状態遷移：CSVに保存されているデータ→任意の新たなデータ
 - 保存形式：保存先ファイル(ID.csv)に追記(ID , 出勤時間　,退勤時間 , 休憩時間)
  [4.一覧]
 - 前提条件：CSVファイルが見つかり、中に出退勤記録が書き込まれている
 - 例外条件：CSVファイルが見つからない場合、CSVファイルに記録が無い場合
 - エラーメッセージ：「」
 - 処理：CSVファイルの内容を参照し、画面に表示
 - 状態遷移：CVS読み取り→画面に一覧を表示
 - 保存形式：保存先ファイル(ID.csv)に追記(ID , 出勤時間)
 [5.終了]
- 前提条件：メニュー画面を起動しているとき
- 例外処理：なし
- エラーメッセージ：なし
- 処理：
    - 半角数字5を入力されたことを読み取った際、「終了します」と表示
    - 任意キーの入力街ちを行い、入力されたらプログラムを終了する
- 状態遷移：起動中→終了
